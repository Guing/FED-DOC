readline的使用方法和实现原理

使用​

readline是Node.js中的一个内置库，主要是用来管理输入流的

const readline = require('readline')

const rl = readline.createInterface({
    input:process.stdin,
    output:process.stdout
})

rl.question('your name:',(answer =>{
    console.log('your name is:'+answer)
    rl.close()
}))
